Write a Python function `slugify(text)` that turns a string into a URL slug: lowercase, with
every run of non-alphanumeric characters replaced by a single hyphen, and no leading or trailing
hyphens. Examples: `"Hello, World!"` -> `"hello-world"`, `"  Foo   Bar  "` -> `"foo-bar"`,
`"C++ Guide"` -> `"c-guide"`. Empty input returns `""`.

Put it in a single Python code block. It will be imported as `from solution import slugify`.
